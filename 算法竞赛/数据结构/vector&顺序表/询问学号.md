# 解题思路
## 整体框架
>可以想象一下，有多个学号，需要从头到尾放到一个新的数组中，如然后再访问对应位置的学号
```
28, 42, 9, 36, 15

将上述的学号放入动态的数组中
vector <int> arr -> [28, 42, 9, 36, 15]

可以访问对应的学号
arr[3] = 9
```
## 做题细节
 有n名同学`n <= 2e6` ， 学号在` 1 ~ e9(说明在整形int范围中)`
# 代码演示
```cpp
#include <iostream>
#include <vector>
using namespace std;

const int num = 2e6 + 10; // 最大数组量

int n, m;
vector<int> a(num); // 创建数组

int main()
{

	cin >> n >> m;
	
	for(int i = 1; i <= n; i++) cin >> a[i];

	int i = 0;
	while(m--)
	{
		cin >> i;
		cout << a[i] << endl;
	}
		
	return 0;
}
```
