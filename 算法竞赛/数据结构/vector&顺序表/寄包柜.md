# 解题思路
## 整体框架
![](图片/寄包柜00.png)
```
第一次输入 `1` 或 `2` 是选择 `存` 或 `取` 的功能

存：就要从5个柜子中选中一个，再从选中的柜子的第n个格子，放入数据

取：就要从5个柜子中选中一个，再访问选中的柜子的第n个格子(前提是有数据)
```
## 做题细节
```
柜子的最大数：const int N = 1e5 + 10;

使用vector：vector<int> a[N]; // 这里表示创建N个柜子

格子不够需及时扩容
if(a[i].size() <= j(需要放入的格子))
			{
				a[i].resize(j + 1);
			}
```
# 代码演示
```cpp
#include <iostream>
#include <vector>
using namespace std;

const int N = 1e5 + 10; // 数组最大值

int n, q;

vector<int> a[N]; // 创建N

int main()
{
	cin >> n >> q;
	while(q--)
	{
		int op, i, j, k;
		cin >> op >> i >> j;
		
		if(op == 1)
		{
			cin >> k;
			if(a[i].size() <= j)
			{
				a[i].resize(j + 1);
			}
			a[i][j] = k;
		}
		else
		{
			cout << a[i][j] << endl;
		}
	}
	
	return 0;
}
```
