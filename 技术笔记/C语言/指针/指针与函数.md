# 1、指针与函数
---
## 1.1函数指针
---
在C语言中，一个函数总是占用一段连续的内存区，而函数名就是该函数所占内存区的首地址。把函数的这个首地址（函数入口地址）赋予一个指针变量，使指针变量指向该函数。然后通过指针变量就可以找到并调用这个函数。我们把这种指向函数的指针变量称为“函数指针变量”。
### 1>语法形式
---
函数指针变量定义的一般形式为：
 ```C
tapy(*变量名)();
int(*pf)();
 ```
 表示pf是一个指向函数入口的指针变量，该函数的返回值（函数值）是整型。
### 2>无参数
---
 ```C
 #define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

void f1()
{
	printf("helloworld\n");
}

int main()
{
	void (*p)(); //定义指针函数
	p = f1;
	p();         //通过函数指针调用函数 等价于 f1()

	return 0;
}
 ```
### 3>有参数和返回值
---
```C
#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>

int add(int x, int y)
{
	return x + y;
}

int main()
{

	int(*p)(int, int) = add;
	printf("%d", p(1, 2));

	return 0;
}
```

也可以使用typedif重命名一个函数指针
```C
#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>

int add(int x, int y)
{
	return x + y;
}

typedef int(*T)(int, int);

int main()
{
	T p1 = add;  //等价于 int(*p)(int,int);
	printf("%d", p1(1, 3));

	return 0;
}
```
## 1.2指针函数
---
-  指针函数是指返回值是指针的额函数。
在C语言中允许一个函数的返回值是一个指针（地址），这种返回指针值的函数称为指针型函数，简称指针函数。注意：不能返回局部变量的地址！！！
### 1>语法形式
---
```
类型说明符 *函数名(形参表)
{
	/*函数体*/
}
```
其中函数名之前家里“ * ”号表明这是一个指针型函数，即返回值是一个指针。类型说明符表示了返回的指针值所指向的数据类型。

	//char str[32] = { 0 };

	//释放内存

## 1.3int(* p)和int* p()的区别？
---
![](图片/QQ20251015-093424.png)
函数和指针结合的两种：
1. 函数指针，定义：int (*p)(),含义：指向函数的指针   
2. 指针函数，定义：int * p()或者例如typedef int* p()，含义：返回值是地址的函数。
# 2、回调函数

## 2.1基础冒泡排序
---
```C
#include<stdio.h>

void swp(int* arr, int len)
{
	for(int i = 0; i < len - 1; i++)
	{
		for(int j = 0; j < len - i - 1; j++)
		{
			if(arr[j] < arr[j + 1])
			{
				//交换
				int t = arr[j];
				arr[j] = arr[j + 1];
				arr[j + 1] = t;
			}
		}
	}
}

int main()
{
	int arr[] = { 4,6,7,1,3,8,3,9,2,4,6,8,4 };
	int len = sizeof(arr) / sizeof(arr[0]);
	
	//交换函数
	swp(arr, len);
	
	for(int i = 0; i < len; i++)
	{
		printf("%d ", arr[i]);
	}
	
	return 0;
}
```
- 只能从小到大打印数字
- 想实现从大到小打印要额外增加代码量
## 2.2回调函数之冒泡排序
---
```C
#define _CRT_SECURE_NO_WARNINGS
#include<stdio.h>

int less(int x, int y)
{
	return (x > y) ? 1 : 0;
}

int greater(int x, int y)
{
	return (x < y) ? 1 : 0;
}


void swp(int* arr, int len, int (*p)(int, int))
{
	int i = 0;
	for (i = 0; i < len - 1; i++)
	{
		int j = 0;
		for (j = 0; j < len - 1 - i; j++)
		{
			// if (arr[j] > arr[j + 1])
			if (p(arr[j], arr[j + 1]))   //通过函数指针调用函数
			{
				//交换
				int t = arr[j + 1];
				arr[j + 1] = arr[j];
				arr[j] = t;
			}
		}
	}
}

int main()
{
	int arr[] = { 4,6,7,1,3,8,3,9,2,4,6,8,4 };
	int len = sizeof(arr) / sizeof(arr[0]);

	//冒泡函数
	swp(arr, len, less);
	
	//打印排序后的数组
	for (int i = 0; i < len; i++)
	{
		printf("%d ", arr[i]);
	}
	printf("\n");

	swp(arr, len, greater);

	for (int i = 0; i < len; i++)
	{
		printf("%d ", arr[i]);
	}

	return 0;
}
```
