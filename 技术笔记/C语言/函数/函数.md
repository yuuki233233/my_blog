# 函数的定义
---
## 函数的组成部分
---
### 语法形式
---
```C
//每个C语言必有的函数
int main()
{
	return 0;
}
//int：返回类型
//main：函数名-->标识符 就是一个名字，想起什么都可以
//()：描述参数
//return 0：函数返回值
```

### 函数返回值类型与返回值
---
#### 1>有返回值
当我们自己写函数时，要具备以下东西
```C
/*
函数返回值类型  函数名(形参)
{
	函数体；
	            //函数返回值类型必须与return后面的返回值一致
	return 返回值
	
}
*/
```

#### 2> 无返回值
还有另一种写法
```C
void main()
{
	//没有return 0;
}
```
void表示没有返回值，就不用加上返回值return。

#### 3>函数传参
函数参数必须要和你定义时候的参数类型一致
```C
void print(int a)//a = 1
{
	if(a == 1)
	printf("lisi");
}
int main()
{
	print(1);//将1传入形参a中
	return 0;
}
```
用图片的方式来展示一下：
![](图片/QQ20251008-085156.png)
打印的结果就是lisi，当类型相同时，传入的实参和形参保持一致。
#### 4>return函数
当我们写成以下代码时
```C
int main()
{
	printf("lisi");
	return 0;
	printf("zhangsan");
}
```
 打印出了lisi，但没打印zhangsan的，因为return是函数的结束标识

#### 5>注意事项
注意点：
1. 函数返回值类型必须与return后面的返回值一致
2. 函数中没有返回值，可以用void表示
3. 函数调用时，函数的实参和形参一致
4. return的作用是结束当前函数

# 函数的定义顺序
---
## 函数放在主函数上面
---
在标准C语言中，函数的定义顺序是有讲究的，默认情况下，只有后面定义的函数才可以调用前面定义过的函数
```C
int add(int a, int b)
{
	return a + b;
}

int main()
{
	int c = add(1,4);
	return 0;
}
```
main函数调用了第1行的add函数，这是合法的。如果调用add函数和main函数的顺序，在标准的C编译器环境下是不合法的。

## 函数放在主函数下面
---
### 函数定义的形式
---
```
返回值类型 函数名(参数1,参数2,...)
```

如果想把其他函数的定义写在main函数下面，而且main函数能正常调用这些函数，拿就必须在main函数的前面作一下函数的声明
```C
//只是做个函数声明，并不用实现
int add(int a, int b);

int main()
{
	int c = add(1,4);
	return 0;
}

//函数的定义(实现)
int add(int a, int b)
{
	return a + b;
}
```
只要你在main函数前面声明过一个函数，main函数就直到这个函数的存在，就可以调用这个函数。

# 函数调用
---
- 调用方式：函数名(函数参数)
	- 函数参数必须和函数定义时候的类型一致
	- 如果不一致，可以转换的不会报错，不能转换就会报错
	- 如果类型不一致，编译器也会报错
 ```C
 //ret是整形类型，函数也要是int类型，返回值也是整形
 int add(int x, int y)//函数形参x接受n值，形参y接受m值
 {
	 return x + y；//整形
 }
 int main()
 {
	 int n = 2;
	 int m = 3;
	 int ret = add(n,m);//传入n，m
 }
 ```
这个我们上面举例过了很多例子，这不过多说

	- 以另一个函数的返回值当做参数嵌套
